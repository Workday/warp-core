WarpJUnitSpec 

trait WarpJUnitSpec extends JUnitSuite with Matchers with TryMatchers with OptionValues with Inside with Inspectors with AssertionsForJUnit

An opinionated starting place.

see http://scalatest.org/user_guide/defining_base_classes
