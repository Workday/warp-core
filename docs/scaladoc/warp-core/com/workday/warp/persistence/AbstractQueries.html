Related Doc: package persistence

trait AbstractQueries extends AnyRef

Defines function definitions for creating read and write Query. These queries can be composed, converted to DBIO, and executed using a Connection to produce results.

In the case where the return value contains some "RowLike", the return type will be DBIO. This is to allow overriding of return types with more specific types. Query is invariant, so it cannot be subclassed, while DBIO is covariant.
